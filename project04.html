Hear n' Now Hip-Hop Podcast and Media Archive

Project Overview


The Hear n' Now Hip-Hop website is a modern, responsive web archive showcasing the podcast episodes, videos, and playlists from Howie and Mickies. Built with Next.js and React, it delivers a smooth browsing experience with dynamic filtering by content type.


The project focuses on clean, minimal design using Tailwind CSS to highlight media content. Users can easily filter posts by podcast, video, or playlist, with clear metadata like duration and date to enhance discoverability.


Social media preview metadata and favicons are carefully implemented for great sharing across platforms. The site is hosted on Vercel for fast, reliable delivery, with a strong emphasis on mobile-first design.


Through this project, I deepened my knowledge of React client components, Next.js routing and layouts, and Tailwind CSS styling, while building a functional media hub for my friends.


thank you, Howie and Mickies :)
